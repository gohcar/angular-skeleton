# Angular skeleton  

## Installation
    git clone git@github.com:santiagohecar/angular-skeleton.git
    cd angular-skeleton
    npm install

## Development
Live rebuild

    npm start
    npm stop
Or 

    npm run build:dev

## Production
    npm run build
