# Angular skeleton  

## Installation
    git clone git@github.com:santiagohecar/angular-skeleton.git
    cd angular-skeleton
    npm install

## Development
    npm run build
    
Live rebuild

    npm start
    npm stop

## Production
    npm run build:prod
